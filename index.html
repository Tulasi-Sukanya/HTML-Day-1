2.Difference between class and Id's:

In HTML, the only difference between class and id is that id's are unique and for each element we have only one id. on the otherhand,class is not unique but we can use same class for multiple elements. In External and internal Css id is represented by '#' and class is represented by '.'.

3.Self Closing Tags:

In HTML, self closing tags are the tags which are closed by itself.(or) It is a type of tag that no need closing tag to end. Ex: br,hr,input,img etc..

4.Types of Headings:

In HTML, we have 6 types of heading tags starting from h1 to h6. h1 to h3 are used to showcase the heading, and h4 to h6 are not much used but we can use them to showcase the subheading.

5.Difference between block level and inline elements:

In HTML, Block level elements always begin on new line,by default block level elements takes some margin. Ex: div, h1 to h6, p, table, form etc..,

Inline Elements does not begin in new line,it only takes the necessary width required for it..Ex:span, img, a etc..

6.Difference between em and strong tags:

In HTML,Strong and em tags are used instead of b and i tags.Strong tag is used to show the importance of the text and em tag is used to show the text in different manner,just to change the voice of tone.
Ex:strong is used instead of bold tag.
em is used instead of italic tag.
